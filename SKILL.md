# 公文排版技能

## 功能概述

本技能用于自动按照国家标准公文格式规范对Word文档进行排版，确保文档符合正式公文的格式要求。

## 排版规范

### 1. 页面设置
- 页边距：上3.7cm，下3.5cm，左2.7cm，右2.7cm
- 纸张：A4

### 2. 字体规范
- **大标题**：方正小标宋_GBK，22磅（二号），居中，无缩进，行间距35磅，**不加粗**
- **一级标题（一、）**：黑体，16磅（三号），加粗，1.13cm缩进，左对齐，行间距28磅
- **二级标题（（一））**：楷体_GB2312，16磅（三号），加粗，1.13cm缩进，左对齐，行间距28磅
- **三级标题（1.）**：仿宋_GB2312，16磅（三号），加粗，1.13cm缩进，左对齐，行间距28磅
- **四级标题（（1））**：仿宋_GB2312，16磅（三号），加粗，1.13cm缩进，左对齐，行间距28磅
- **正文**：仿宋_GB2312，16磅（三号），1.13cm缩进，**左对齐**，行间距28磅
- **图片说明**：楷体，14磅（四号），居中
- **英文和数字**：Times New Roman字体

### 3. 缩进说明
- **首行缩进**：1.13cm（实际测量值）
- 左对齐：所有正文段落左对齐（不是两端对齐）

## 任务流程

1. **读取文档**：首先读取用户提供的Word文档
2. **备份原始文档**：在处理前备份原始文档，以防数据丢失
3. **格式化处理**：根据规范进行格式化
4. **保存文档**：保存排版后的文档
5. **提供总结**：详细描述操作过程和格式化内容

## 使用方法

### 基本使用

当你需要对Word文档进行公文排版时，使用以下步骤：

1. 首先检查是否安装了python-docx库：
   ```bash
   pip install python-docx
   ```

2. 使用以下命令进行排版：
   ```bash
   python -m skills.gongwen.scripts.format_doc <input_file>
   ```

### 支持的文件格式

- 支持.docx格式的Word文档
- 不支持.doc格式的旧版Word文档

## 代码实现

### 脚本位置

排版脚本位于 `scripts/format_doc.py`

### 核心功能

脚本包含以下主要功能：

1. **页面设置**：设置正确的页边距
2. **字体格式化**：自动识别标题和正文，设置相应字体
3. **缩进控制**：设置正确的段落缩进
4. **对齐方式**：标题左对齐，正文两端对齐，图片说明居中
5. **行间距**：设置正确的行间距

## 注意事项

1. **字体依赖**：需要系统支持方正小标宋_GBK、黑体、楷体_GB2312、仿宋_GB2312等字体
2. **环境要求**：需要Python 3.6+和python-docx库
3. **文件备份**：处理前会自动备份原始文档
4. **格式识别**：基于文本内容识别标题级别，可能需要手动调整复杂文档
